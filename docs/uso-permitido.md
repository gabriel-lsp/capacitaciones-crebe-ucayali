# Uso permitido

Este documento orienta el uso permitido del módulo **Capacitaciones Accesibles - CAP**, parte del **Ecosistema Virtual Accesible (EVA)**.

El proyecto tiene una finalidad educativa, formativa, accesible y no comercial. Puede ser consultado y utilizado como recurso de apoyo en procesos de capacitación, orientación, sensibilización o fortalecimiento de capacidades.

## Usos permitidos

Se permite el uso del repositorio para:

- Consultar materiales formativos.
- Organizar recursos de capacitación.
- Apoyar acciones de orientación docente o familiar.
- Difundir recursos educativos vinculados a inclusión y accesibilidad.
- Conservar evidencias digitales de materiales formativos.
- Tomar como referencia la estructura digital del proyecto, respetando los créditos.

## Condiciones de uso

Toda reutilización debe conservar la referencia al proyecto, al autor responsable y a las fuentes externas empleadas en cada material.

## Límites de uso

No se debe presentar el proyecto como propio, eliminar créditos o publicar versiones modificadas sin diferenciar claramente los cambios realizados.

El uso comercial de contenidos, diseños, grabaciones, documentos, flyers, diapositivas o materiales vinculados al repositorio requiere autorización correspondiente.

## Materiales externos

El código del repositorio se rige por el archivo `LICENSE`. Esta licencia se aplica al software y a la estructura técnica, pero no autoriza automáticamente el uso libre de videos, flyers, infografías, diapositivas, documentos externos, materiales educativos o recursos elaborados por terceros.

Los materiales externos mantienen sus propias condiciones de autoría, uso, difusión y reproducción.
