# Fuentes y créditos

Este documento registra los criterios de reconocimiento de fuentes y créditos del módulo **Capacitaciones Accesibles - CAP**, parte del **Ecosistema Virtual Accesible (EVA)**.

El proyecto es desarrollado y organizado por Gabriel Berrospi como recurso educativo digital personal orientado a la formación, accesibilidad, inclusión educativa y atención a la diversidad.

## Criterios generales

Todo material incorporado al repositorio debe conservar su fuente, autoría o procedencia cuando corresponda.

La organización de recursos, adaptación digital, estructura de navegación, criterios de publicación y disposición formativa corresponden al trabajo realizado para este proyecto, salvo que se indique expresamente una fuente externa.

## Recursos externos

Cuando se incorporen flyers, videos, diapositivas, infografías, fotografías, documentos, enlaces, grabaciones, logos o materiales descargables, deberá registrarse su procedencia y respetarse sus condiciones de uso.

Si se utilizan materiales provenientes de fuentes públicas, repositorios abiertos, plataformas formativas, bancos de imágenes o autores externos, se debe mantener la referencia a la fuente original cuando corresponda.

## Datos recomendados para registrar una fuente

Se recomienda registrar cada recurso externo con los siguientes datos:

- Nombre del recurso.
- Autor, fuente o responsable.
- Enlace o procedencia.
- Tipo de recurso.
- Condición de uso o licencia, si está disponible.
- Fecha de consulta o incorporación.
- Capacitación o actividad en la que fue utilizado.

## Reconocimiento del proyecto

Toda reutilización del repositorio debe conservar el reconocimiento del proyecto y del autor responsable.
